# Todo App
Simple Todo Web App manages user todo list.

## Features
1. Add Todo item.
2. Display All Todo Items.
3. Mark Todo Item as Completed.
4. Delete Todo Item.
5. Sort Items based on their status(completed, incomplete).
6. Display the completed and total Todo Item count.
